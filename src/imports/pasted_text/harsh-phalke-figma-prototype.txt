SECTION 3A — EVENT SHOOT SELECTOR

Each event card checkbox → toggles selected state ON/OFF

ON state: card gets gold border highlight, checkbox filled
OFF state: card returns to default grey


Full Day / Half Day toggle on each card → switches pricing label shown on card

Full Day selected: show full day price
Half Day selected: show half day price


Selecting any event → automatically updates the Live Bill in Section 4


SECTION 3B — ALBUM PACKAGES

8×24 card → click to select (highlighted state)
12×36 card → click to select (highlighted state)
Only one album selectable at a time (radio button behavior)

SECTION 3C — ADD-ON SERVICES

Each add-on service card toggle → ON/OFF

ON: card highlighted, price added to bill total
OFF: price removed from bill total


Live running total at bottom of add-ons section → updates as toggles change

Customer Name field → typing updates the name shown on the bill card

"Back / Edit" button on bill overlay → returns to bill generator
